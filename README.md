# RISC-V Banking System

A mini banking system implemented in RISC-V assembly language, simulated using the Ripes RISC-V simulator.

## 📌 About

This project was built as part of our Computer Organization and Architecture (COA) coursework. It simulates a basic banking terminal using low-level RISC-V assembly instructions.

## ✨ Features

- 💰 Deposit funds
- 💸 Withdraw funds (with insufficient funds error handling)
- 📊 Check account balance
- ✅ Transaction success confirmation
- ❌ Error handling: Insufficient Funds
- 👋 Exit with farewell message

## 🛠️ Tools Used

- **Language:** RISC-V Assembly
- **Simulator:** [Ripes](https://github.com/mortbopet/Ripes)

## 📁 Files

| File | Description |
|------|-------------|
| `banking.s` | Main assembly source code |
| `banking.asm` | Assembly file |
| `banking.bin` | Compiled binary |
